fn try_to_nominal<N>(self, system: &N) -> Result<NominalString, Error<Self>>
where N: NominalSystem<Self> + ?Sized,

Tries to format self as a nominal identifier using system.

Errors

Each nominal system can use its own error type. The crate-level error type is Error, and each variant describes why formatting a nominal may fail.
